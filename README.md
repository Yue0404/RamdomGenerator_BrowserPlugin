# 随机数生成器 (Random Generator)

一个功能丰富的 Chrome/Edge 浏览器扩展，提供多种随机数生成能力。

## 功能

| 模式 | 说明 |
|------|------|
| **小数** | 生成 0–1 之间指定小数位的随机小数（最高 32 位） |
| **整数** | 在自定义范围内生成随机整数 |
| **口令** | 生成密码学安全的随机口令，可控长度、字符类型、特殊字符权重 |
| **颜色** | 随机颜色，支持 RGB / HEX / HSL / LAB / CMYK + 透明度控制 + 简约/高级模式 |
| **随机序列** | 批量生成随机数数组（支持小数/整数切换，最多 1024 个） |

## 特性

- **口令保密**：口令保密级别可调，CSPRNG 安全随机
- **历史记录**：支持导出 JSON
- **偏好记忆**：所有参数跨会话持久化
- **硬数据校验**：非法输入实时红框告警
- **一键复制**：全局复制 + 各字段单独复制

## 安装

1. 克隆或下载本仓库
2. 打开 Chrome，进入 `chrome://extensions`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目根目录，完成

## 项目结构

```
├── manifest.json       # Chrome Extension Manifest V3
├── popup.html          # 弹出窗口 UI
├── popup.css           # 样式表
├── popup.js            # 核心逻辑
├── icons/              # 扩展图标 (16/48/128)
└── README.md
```

## 技术栈

- **Manifest V3**，纯前端，零外部依赖
- `crypto.getRandomValues()` — 口令生成使用 CSPRNG
- `chrome.storage.local` — 设置 & 历史持久化
- CIELAB D65 / CMYK 颜色空间转换
- FLIP 动画
